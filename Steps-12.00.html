Step Name : Create Dax Measure Table 12

?

1.From Power BI desktop ribbon, Home tab click on Enter Data.
2.Name the table ‘ Measures/All_Measures’ and the column ‘Measures’ as well. 
3.Start creating measures after right clicking on the column measure and then selecting ‘New measure’. 
4. Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select Commit (checkmark icon) in the formula bar. 
5. After creating each measure choose the necessary format and decimal for each value (measure) then save. 
Follow the gif and create the table and create the measures as columns in the All_Measures table.

GPM% 
= (DIVIDE(SUM('fortune_global_500_from_2019-20'[profits]),SUM('fortune_global_500_from_2019-20'[revenues])))

GPM_YOY% 
= VAR LY_GPM =
CALCULATE(DIVIDE(AVERAGE('fortune_global_500_from_2019-20'[profits]),
AVERAGE('fortune_global_500_from_2019-20'[revenues])),DATEADD('Calendar'[Date], -1, YEAR))
RETURN IF(ISBLANK(LY_GPM),0,DIVIDE(DIVIDE(
            AVERAGE('fortune_global_500_from_2019-20'[profits]),
            AVERAGE('fortune_global_500_from_2019-20'[revenues])) - LY_GPM, LY_GPM))
