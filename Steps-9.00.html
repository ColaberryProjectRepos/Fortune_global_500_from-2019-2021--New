Step Name: Transforming the data in power Query Editor 9
  • In this step you will clean and transform the data
  • Select Transform Data from Home tab.
  •  Transform the Year column:
  •  Duplicate Year column and rename the duplicate column ‘End of Year’. 
  •  Set the datatype for this column as Date. Then select the column and transform it into ‘Year’ and then ‘start of Year’. 
  •  In the Home ribbon, go to Transform and change the Data Type to Text and then to Date. Select Add New Step. 
  • Right click on the End of Year column, select Transform, Year, and then start of Year. The data type will be set to Date. 
  •  Select the  assets ,revenues, profits column and change the datatype to Fixed Decimal number. Then click on the new column and choose custom column and then choose the column that you want it to add million by multiblied (1000000) , same step for profits column and revenues column.
  •  Click on file and select Close and Apply.
