Step Name: Download and Load Excel File to Power BI Desktop 8

Dataset Links:

https://www.kaggle.com/datasets/prasertk/fortune-global-500-from-20192021

1.     Download dataset from Kaggle and save it under your desktop.

2.      Open Power BI Desktop

3.      Click on Get data button. 

4.      Select excel workbook and click on connect.

5.     Select the fortune_global_500_from (2019-2021)  file and click on Open

6.      A preview of the data will be displayed, click on Load/Transform
